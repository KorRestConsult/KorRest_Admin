# KorRest

Личный кабинет KorRest для управления клиентами и подключенными заведениями.

## Что есть в первом MVP

- вход владельца по PIN;
- список клиентов и заведений;
- статусы: работает, подключение, пауза;
- карточка клиента с контактами, тарифом, ссылками проекта и таблицы;
- базовая статистика по клиентам;
- журнал действий;
- облачная синхронизация через Firebase/Firestore;
- локальное хранение в браузере как резервная копия.

## Запуск локально

```bash
python3 -m http.server 8090
```

Открыть:

```text
http://localhost:8090
```

## Firestore

Первый облачный документ:

```text
mirra/korrestAdmin
```

Это быстрый совместимый путь под текущие правила Firestore. Данные KorRest не смешиваются с состоянием MIRRA: это отдельный документ кабинета владельца.
